    – making a copy of a slice before appending it to another slice – two techniques that work and perform the same – make a copy and append it

      output = [][]int{}
  for {
    // doing something here that will change the elements of curr
    cp := make([]int, len(curr))
    copy(cp, curr)
    output = append(output, curr)
  }

    can also do this, these two

      output = [][]int{}
  for {
    // doing something here that will change the elements of curr
    output = append(output, append([]int, curr...)) // copy and append in one line
  }
